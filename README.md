# Wafer-Joining-Analysis-Optimization
Uncovering Persona of Joining &amp; Non-Joining Wafer in Highly Overlapping Sensor Data

## Quick Summary
* **Objective:** Identify the "Golden Zone" for wafer joining to reduce defects.
* **Challenge:** High data overlap between joining and non-joining status.
* **Methodology:** Used PCA for feature extraction and Quartile Analysis to set threshold limits.
* **Impact:** Defined an Operating Window (Vibration & UV) that correlates with a 79% success rate.

Dataset Source: [(https://www.kaggle.com/datasets/programmer3/semiconductor-sensor-data-for-predictive-quality)]
